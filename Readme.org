* XMonad installation

To install xmonad on Ubuntu 22.04 I use ghcup.

For building install ~ghc 9.4.7~. newer ghc version will fail to build
due to dbus requiring ~bytestring<0.12~ and newer ghc ships
~bytestring 0.12~.
